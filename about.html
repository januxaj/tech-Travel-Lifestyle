About Me

Hello! I'm Edwine Nyakundi, a Full Stack Developer and Machine Learning Enthusiast from Kenya, currently pursuing a Bachelor's degree in Computer Science at Karatina University (2021–2025).

I’m passionate about crafting intelligent and efficient software. I specialize in full stack development (React, Node.js, Spring Boot, Django), mobile apps (React Native, Android SDK), and machine learning using Python libraries like TensorFlow, Scikit-learn, and Pandas.

Over the years, I've worked with startups and remote teams such as Travelsjinn (India), SkillFirm, and I founded my own platform called Marktday. My roles have involved everything from WordPress plugin development to deploying predictive models and building REST APIs.

My standout projects include:

  • Customer Churn Predictor – helping telecom companies retain users using ML.
  • HealthTrack – a fitness tracker app with data-driven coaching.
  • SmartBudget – a React Native app that provides smart financial insights.
  • TravelBlog – a complete WordPress-based travel content platform.

I enjoy blending creativity with logic — designing smooth user experiences while writing robust backend code. When I'm not coding, I’m learning new tech, exploring travel destinations, or mentoring junior developers online.
